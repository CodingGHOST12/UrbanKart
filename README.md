# UrbanKart - Indian Grocery E-commerce App

UrbanKart is a modern Indian grocery e-commerce web application built using React + Vite. The application provides a clean shopping experience with realistic Indian grocery products, cart management, checkout flow, search functionality, Redux state management, and responsive UI.

---

# Live Features

- Product listing page
- Product detail page
- Shopping cart management
- Quantity increase/decrease
- Remove from cart
- Checkout functionality
- Search products
- Responsive modern UI
- Redux Toolkit state management
- React Router dynamic routing
- Lazy loading
- Error handling
- 404 Not Found page
